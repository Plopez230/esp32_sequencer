## A toy full of romanticism
- YM2413 sound chip from old game cartridges.
- Capacitive keyboard from aluminum envelopes.
- Plastic bags.
- Cardboard boxes from the neighbors' new television.
- Broken USB cables.
- LCD screen, SD card reader.
